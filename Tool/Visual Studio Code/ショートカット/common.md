# カーソル移動
## エクスプローラーに移動
- [【小ネタ】極力キーボードから手を離したくないライトvimmerがVSCodeに設定しているショートカット（基本的なファイル操作編）](https://dev.classmethod.jp/articles/vscode_file_operation_shortcut_setting/)
